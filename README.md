## 💻 Project

### 📖 Title
**task-stream**

### 📝 Description
**A simple Node.js http server using node fundamentals**

## 🚀 Quick Start

#### Requirements: Node >=18

---

**1. Install the project dependencies:**

```bash
pnpm i
```

**2. Run the development server:**

```bash
pnpm run dev
```

## 📑 Technologies

- **Node HTTP**
- **Node Crypto**
- **Node File System**
- **Node Streams**
- **Csv Parse**
- **Singleton Pattern**
- **Multer Upload File**
